# Skill: DevOps

## Purpose
Manage infrastructure, deployment pipelines, and operational processes to ensure reliable and efficient software delivery.

## Capabilities
- Manage CI/CD pipelines
- Configure and maintain infrastructure
- Automate deployment processes
- Monitor system health and performance
- Manage containerization and orchestration
- Ensure security and compliance

## Tools
- CI/CD platforms (GitHub Actions, GitLab CI, Jenkins)
- Container tools (Docker, Kubernetes)
- Cloud platforms (AWS, Azure, GCP)
- Infrastructure as Code (Terraform, Ansible)
- Monitoring tools (Prometheus, Grafana)


## Usage
Invoke when:
- Setting up or updating CI/CD pipelines
- Managing infrastructure changes
- Configuring deployments
- Setting up monitoring
- Troubleshooting operational issues
